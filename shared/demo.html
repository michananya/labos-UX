Filter

labos-filter — overlay panel for saved filters gallery, active filter chips, and filter actions. Opens under the navbar with a semi-transparent backdrop.

With active filter
Event log: —

API
<!-- Declare anywhere in body -->
<labos-filter id="my-filter"></labos-filter>

<!-- Trigger button (placed last in toolbar) -->
<labos-button icon="filter_list">Filter</labos-button>

<script>
  const f = document.getElementById('my-filter');

  // Populate saved filters gallery
  f.setFilters([
    { name: 'Active orders', count: 45, value: 'active' },
    { name: 'Pending review', count: 12, value: 'pending' },
  ]);

  // Show active filter with chips
  f.setActiveFilter({
    name: 'Active orders',
    chips: [
      { label: 'Status', value: 'Active', key: 'status' },
      { label: 'Age',    value: '> 30', key: 'age'   },
    ]
  });

  f.clearActiveFilter();  // hide row 2

  // Toggle on button click
  btn.addEventListener('click', () => f.toggle());

  // Events
  f.addEventListener('labos-filter-select',      e => console.log('selected', e.detail.value));
  f.addEventListener('labos-filter-action',       e => console.log('action', e.detail.action));
  f.addEventListener('labos-filter-chip-remove',  e => console.log('chip removed', e.detail.value));
  f.addEventListener('labos-filter-settings',     () => console.log('go to filter management'));
  f.addEventListener('labos-filter-search',       e => console.log('search', e.detail.query));
  f.addEventListener('labos-open',  () => console.log('opened'));
  f.addEventListener('labos-close', () => console.log('closed'));
</script>

Line chart

labos-line-chart — Highcharts-based line chart with LabOS design tokens. Supports multi-series data, x-axis categories, shared tooltip, and point click events. Highcharts is loaded on demand from CDN.

Multi-series
Single series · no legend
Events — click a point
No point clicked yet
Usage
<labos-line-chart id="chart" title="Title" y-title="Y label" height="400"></labos-line-chart>
<script>
document.getElementById('chart').setData(
  [
    { name: 'Series A', data: [1.2, 3.5, 2.8, 4.1, 5.0] },
    { name: 'Series B', data: [0.8, 2.1, 3.3, 2.9, 4.4] },
  ],
  ['Mon', 'Tue', 'Wed', 'Thu', 'Fri']
);
</script>
Attributes
Attribute Default Description
title — Chart title text
height 400 Chart height in pixels
y-title — Y-axis label
legend true "false" to hide the series legend
